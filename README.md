# 🔢 Number Checker Form (DOM)

A simple and responsive **Number Checker Form** built with **HTML5, CSS3, Bootstrap 5, and JavaScript (DOM)**. This project checks whether a user-entered number is **Even** or **Odd** using JavaScript DOM manipulation and displays the result instantly.

## 📖 How It Works

1. Enter a number.
2. Click the **Check** button.
3. The program determines whether the number is:
   - Even
   - Odd
4. The result is displayed immediately without refreshing the page.
